How to connect two devices:

  1. Open this page on two different devices
  2. On both devices, click "Start Camera"
  3. Person A: Click "Create Call" and share the call code with Person B
  4. Person B: Paste the call code and click "Process Code" to join
  5. Person B: Copy the answer code and send it back to Person A
  6. Person A: Paste the answer code and click "Process Code"
  7. You should now be connected!
